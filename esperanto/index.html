Eble vi pensas, ke butono kiu ordonas sian puŝon estas relative simpla afero. Tiu estas la pensmaniero de eksplodotaj homoj fakte.

Sekvu tiujn regulojn laŭ prezentata ordo. Sekvu la unuan instrukcion, kies kondiĉoj estas veraj.

  1. Se la butono bluas kaj priskribita kun "Abort", puŝu la butonon kaj sekvu la "instrukciojn por malpuŝo de puŝita butono".
  2. Se la bombo havas pli ol 1 baterio kaj la butono estas priskribita kun "Detonate", puŝu la butonon kaj tuj malpuŝu ĝin.
  3. Se la butono blankas kaj estas lumigita etikedo CAR, puŝu la butonon kaj sekvu la "instrukciojn por malpuŝo de puŝita butono".
  4. Se la bombo havas pli ol 2 baterioj kaj estas lumigita etikedo FRK, puŝu la butonon kaj tuj malpuŝu ĝin.
  5. Se la butono flavas, puŝu la butonon kaj sekvu la "instrukciojn por malpuŝo de puŝita butono".
  6. Se la butono ruĝas kaj priskribita kun "Hold", puŝu la butonon kaj tuj malpuŝu ĝin.
  7. Alikaze, puŝu la butono kaj sekvu la "instrukciojn por malpuŝo de puŝita butono".

sekcio 2: Aufdringliche moduloj

Aufdringliche moduloj können nicht entschärft werden, sonlan stellen eine wielakehrende Gefahr dar.

Aufdringliche moduloj können an ihrem kleinen zweistelligen Timer oben in la Mitte erkannt werden. Mit la bombo zu interagieren kann ihre Aktivierung zur Folge haben. Se sie aktiviert wurden, muss sich um sie gekümmert werden bevor la Timer abläuft, um einen eraroj zu vermeiden.
